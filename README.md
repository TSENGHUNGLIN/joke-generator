# 隨機笑話產生器

這是一個簡單的靜態網頁專案，可以在瀏覽器中隨機顯示繁體中文笑話，並支援複製與語音朗讀。

## 功能

- 取得隨機繁體中文笑話
- 複製笑話到剪貼簿
- 使用瀏覽器語音合成朗讀笑話

## 使用方式

1. 直接開啟 `index.html`
2. 按下「取得笑話」取得新的笑話
3. 按下「複製笑話」可複製文字
4. 按下「朗讀笑話」可由瀏覽器朗讀全文
